# 🗺️ Éditeur de fichiers GPX

Application web pour modifier des fichiers GPX (traces GPS) :
couper un itinéraire, supprimer une portion, ajouter/déplacer
des waypoints, redéfinir le point de départ d'une boucle.

## 🚀 Installation

```bash
git clone https://github.com/TON_PSEUDO/editeur-gpx.git
cd editeur-gpx
python -m venv venv
source venv/bin/activate  # Sur Windows : venv\Scripts\activate
pip install -r requirements.txt
